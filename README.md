# Just Tech News
## Description 
A full-stack application that allows users to sign in/up, create posts, vote on other users' posts, and leave comments. <br />
Built With: <br />
* Handlebars
* CSS
* JavaScript
* Express.js
* Node.js
* MySQL
* Sequelize
* dotenv
* bcrypt
